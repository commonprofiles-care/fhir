Introduction

This ImplementationGuide is published by a collaborative group of Swedish vendors of healthcare IT systems. It includes implementation profiles, extensions and value sets created by the group. It also includes core profiles, value sets and extensions which have not been published by this group but are used in one way or another in the implementation profiles. One example is the Swedish base profiles which have been derived to implementation profiles published in this ImplementationGuide.
